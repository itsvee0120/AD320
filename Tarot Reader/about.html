This web app is a simple tarot card reader that generates a random card from the tarot deck. The user can click the "Generate Card" button to see a new card. User can also search for a card using the tarot dictionary function.

For this Tarot card web app, I chose the Tarot API for its comprehensive data on tarot cards, including their meanings and details. The app features two main functions: drawing a random card, which displays a card's information, and searching for specific tarot cards by name. Challenges included handling asynchronous API calls using the Fetch API and ensuring a responsive design with a collapsible menu for navigation. Despite these challenges, the app delivers an interactive and user-friendly tarot experience.
